## About The Project

Udra Kagaz Innovations is a social enterprise focused on the circular economy. This project is the public-facing platform designed to pitch to University Campuses and CSR (Corporate Social Responsibility) Sponsors. 

The platform highlights UKI's two core divisions:
1. **PUNARJEEVAN (Reuse):** Extracting blank pages from discarded lab records, binding them into notebooks, and donating them to underprivileged children.
2. **NIRMAAN (Recycle):** Pulpng the remaining written scrap to produce premium handcrafted paper products (diaries, card stock) to sustain operations.

### Key Features
- **Interactive Waste Calculator:** A dynamic tool designed for university administrators to input their student population and instantly calculate their annual paper waste, lost revenue, and environmental impact.
- **Nirmaan Store:** A beautiful eCommerce grid layout showcasing the handcrafted, upcycled paper products.
- **Zero-Dependency Architecture:** Built entirely in Vanilla HTML, CSS, and JS for maximum performance, instantaneous load times, and easy hosting.
- **Glassmorphism Design:** Modern UI/UX utilizing 'Forest Dark' themes to match the environmental mission.

## Tech Stack

*   **HTML5**
*   **Vanilla CSS3** (Custom Grid, Flexbox, Keyframe Animations)
*   **Vanilla JavaScript (ES6)** (Intersection Observers, Dynamic DOM Math)

## Getting Started

Because this is a vanilla front-end project, there are no complex build steps or dependencies required.

### Prerequisites

You simply need a modern web browser.

### Installation

1. Clone the repo
   ```bash
   git clone https://github.com/SOUMYA-soumya/UDRA_KAGAZ_WEBAPP.git
   ```
2. Open `index.html` in your browser.
   ```bash
   # On Windows
   start index.html
   
   # On Mac
   open index.html
   ```

*Alternatively, you can run a local server:*
```bash
python -m http.server 8000
# Then navigate to http://localhost:8000
```
